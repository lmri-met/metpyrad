File structure#

The file is structured with header information about the measurement cycle followed by a series of repeated blocks of information for each background/sample measurement.

Cicle ID information
Block 1:   Repetition 1, background measurement
Block 2:   Repetition 2, background measurement
...
Block n:   Repetition n, background measurement
Block n+1: Repetition 1, sample measurement
Block n+2: Repetition 2, sample measurement
...
Block n+n: Repetition n, sample measurement

File contents#

Header#

The header information contains information about the measurement cycle, like a title, start time, ROI channel limits, and counting type.

1Lu-177 HS3 301123_ciclo1
2Start Time 8:40:58
3- ROI1 Free Channel Limits 1 - 1023, Type Beta
4Counting type: Low

Blocks#

In the series of repeated blocks of information, the key words “Sample start” indicate the beginning of a block of background/sample measurement information. Each block is structured with sample details (sample number, repetition, vial…), measurement data (count rate, counts, time parameters, quenching parameters…) and spectrum data for alpha and beta particles.
